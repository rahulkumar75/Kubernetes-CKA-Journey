Git daily command reference cheat sheet

S
Snapshot
git statuswhat's changed
git add <file>stage file
git add .stage everything
git add -pstage by hunk
git diff --stagedreview staged
git commit -m "msg"commit
git commit --amendedit last commit
git commit --amend --no-editadd to last commit
B
Branches
git branchlist branches
git switch <branch>checkout
git switch -c <branch>create + checkout
git branch -d <branch>delete (safe)
git branch -D <branch>force delete
git branch -m <old> <new>rename
M
Merge & Rebase
git merge <branch>merge in
git merge --no-ff <branch>preserve history
git merge --abortcancel merge
git rebase <branch>rebase onto
git rebase -i HEAD~Ninteractive rebase
git cherry-pick <hash>apply commit
R
Remote
git remote -vlist remotes
git fetchfetch all
git pullfetch + merge
git pull --rebasefetch + rebase
git pushpush branch
git push -u origin <branch>push + track
git push --force-with-leasesafe force push
U
Undo
git restore <file>discard changes
git restore --staged <file>unstage
git revert <hash>safe undo commit
git reset --soft HEAD~1undo, keep staged
git reset --mixed HEAD~1undo, keep files
git reset --hard HEAD~1⚠ erase commit
git clean -fd⚠ remove untracked
T
Stash
git stashsave dirty state
git stash push -m "name"named stash
git stash listlist stashes
git stash popapply + drop
git stash apply stash@{n}apply, keep stash
git stash drop stash@{n}delete stash
H
History & Search
git log --oneline -10compact log
git log --graph --onelinebranch graph
git log -p <file>file history
git show <hash>inspect commit
git blame <file>who wrote what
git bisect startbinary search bug
git reflogfull action log
C
Config & Init
git initnew repo
git clone <url>clone remote
git config --global user.nameset name
git config --global user.emailset email
git config --listshow config
git tag <v1.0>create tag
git push origin --tagspush tags
Tips git diff HEAD — all unstaged changes  ·  HEAD~2 — 2 commits back  ·  git log --author="name" — filter by author  ·  git push --force-with-lease over --force to avoid overwriting teammates' work
